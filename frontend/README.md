# PulseBoard Frontend

PulseBoard is a modern project management and incident tracking dashboard built with React, TanStack Router, and TanStack Query. It provides a clean, responsive interface for managing projects, tasks, users, and incidents in real time.

## Features

- **Project Management** – Create and organize projects with task tracking
- **Incident Tracking** – Monitor and manage incidents across projects
- **User Management** – Administer team members and permissions
- **Real-time Updates** – Powered by TanStack Query for seamless data synchronization
- **Dark/Light Theme** – Switch between themes with a single click
- **Responsive Design** – Works on desktop and mobile devices

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Tech Stack

- **React** – UI library
- **TanStack Router** – Type-safe routing
- **TanStack Query** – Data fetching and caching
- **Tailwind CSS** – Utility-first styling
- **Vite** – Fast build tooling
